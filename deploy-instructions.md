# Quick Deploy to Vercel

## Steps:

1. Login to Vercel:
```bash
npx vercel login
```

2. Deploy to production:
```bash
npx vercel --prod
```

3. Follow the prompts:
   - Link to existing project? N
   - Project name: natan-equity-research  
   - Directory: ./ (default)
   - Override settings? N

4. Get your live URL!

---

## Alternative: GitHub Integration (Automated Deployments)

1. Push your code to GitHub
2. Go to vercel.com
3. Import your GitHub repo
4. Every push auto-deploys!
